# grajek
Prosty grajek do strumieni internetowych

<img width="302" height="425" alt="image" src="https://github.com/user-attachments/assets/d1ee116a-56af-4e4e-a0a0-9a3615755193" />

- Niewielkich rozmiarów plik wykonywalny (poniżej 400 kB),
- mikroskopijne wykorzystanie CPU i RAM,
- brak instalatora - po prostu rozpakuj archiwum zip i uruchom `grajek.exe`,
- wszystkie ustawienia są przechowywane w folderze programu, w pliku `prefs.ini`

Odtwarzanie różnych formatów możliwe dzięki wykorzystaniu biblioteki [BASS](https://www.un4seen.com/).
